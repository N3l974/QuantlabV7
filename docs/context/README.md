# Context Hub (Lightweight)

Ce dossier centralise le **contexte actif** pour éviter de charger trop d'historique dans chaque session.

## Fichiers

- `ACTIVE_CONTEXT.md` : état courant minimal (à garder court)
- `SESSION_TEMPLATE.md` : template de démarrage de session
- `ARCHIVE_INDEX.md` : index des documents archivés / lourds

## Règles

1. `ACTIVE_CONTEXT.md` doit rester < 120 lignes.
2. Ne pas recopier des logs complets ici.
3. Mettre les détails historiques dans `docs/archive/`.
4. Référencer les fichiers lourds par lien, pas par copier-coller.
