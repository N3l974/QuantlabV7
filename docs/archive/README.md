# Archive Documentation

Ce dossier est réservé aux contenus peu utiles au quotidien mais importants pour la traçabilité.

## Contenu recommandé
- anciens plans
- comptes rendus de sessions longues
- rapports expérimentaux non actifs
- brouillons de docs remplacées

## Bonnes pratiques
- Préfixer les fichiers avec une date : `YYYYMMDD_nom.md`
- Ajouter une ligne `Status: archived` en haut du fichier
- Lier chaque archive depuis `docs/context/ARCHIVE_INDEX.md`
