# Portfolio Archive — Historical Versions

Ce dossier contient les artefacts des versions historiques de portfolios, organisés par version.

## Structure

- `v1/` — Meta‑optimisation, Portfolio V1, diagnostics multi‑asset
- `v2/` — Portfolio V2 avec leverage
- `v3/` — Portfolio V3 et V3b (concentration ETH)
- `v4/` — Diagnostic V4, edge, audit, tests, Portfolio V4

Chaque sous‑dossier contient les rapports markdown et les JSON associés à cette version.

## Règle

Ces artefacts sont **archivés** et ne doivent pas être modifiés. Ils servent de référence historique pour la traçabilité des évolutions.
