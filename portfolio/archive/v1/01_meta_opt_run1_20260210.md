## Diagnostic

### Anomalies critiques

1. **Sharpe élevé + PF < 1 = impossible en réalité**
   - Un Sharpe > 1 signifie que la stratégie gagne plus qu'elle ne perd (ajusté au risque)
   - Un PF < 1 signifie que les pertes totales > gains totaux
   - Ces deux métriques sont contradictoires → **bug confirmé dans le backtester**

2. **Returns irréalistes** (47398x, 20849x, etc.)
   - Avec $10K de capital initial, un return de 47398x = $474M
   - Impossible sur BTC 1h avec des commissions de 0.1%
   - Cause : double-comptage de l'equity (cash + position value comptés deux fois)

3. **Concentration sur bollinger_breakout 1h**
   - ~65% des profiles sont bollinger_breakout sur 1h
   - Probablement parce que le bug de double-comptage amplifiait les returns sur les TFs rapides

## Conclusion

**Run entièrement invalide.** Les résultats ne reflètent pas la réalité du marché. Le backtester a été corrigé après cette run (voir corrections du 11 février).
